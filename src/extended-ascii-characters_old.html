Extended ASCII Characters
(128-255)

ASCII stands for American Standard Code for Information Interchange. The extended characters are an 8-bit coded character set consisting of control codes and printable characters. The extended characters were created once a demand for non-english letters developed. Most West European language characters were added. The range 128-159 is the C1 control code set, which is an extension from the C0 control code set used with the standard US-ASCII. Currently, the range 128-159 is mostly used for printable characters. Windows-1252 and ISO 8859-1 are equal in the 160-255 range. It is referred to as the Latin-1 Supplement Unicode characters.
