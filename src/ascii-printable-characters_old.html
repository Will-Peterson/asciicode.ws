ASCII Printable Characters
(32-126)

ASCII stands for American Standard Code for Information Interchange. The printable characters are a 7-bit character set containing numbers, punctuation characters, and the Latin upper and lower case letters. It is also considered the Basic Latin character set. The first 128 ASCII symbols (0-127) was incorporated into the Unicode character set. This allows backward compatibility with UTF-8 (Unicode Transformation Format 8-bit).
