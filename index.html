ASCII Control Characters
(0-31 & 127)

ASCII stands for American Standard Code for Information Interchange. The control characters are referred to as the C0 control code set. It is 7-bit invisible character set that does not represent written symbols. They are called control codes because it is possible to generate a code using the control (Ctrl) key plus another key. The control characters are rarely used any more but they are still part of ISO 8859-x and windows-125x. ISO 646 defines the C0 control code set character references and descriptions.
